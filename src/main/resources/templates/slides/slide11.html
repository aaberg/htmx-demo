Forms with inline validation

Html with htmx
<h4>Signup form</h4>
<form th:hx-post="@{/signup}">
    <div hx-target="this" hx-swap="outerHTML">
        <label>Email
            <input name="email"
                hx-post="/signup/email-validation"
                value="" />
        </label>
    </div>
    <div>
        <label>First name
            <input name="firstName" />
        </label>
    </div>
    <div>
        <label>Last name
            <input name="lastName" />
        </label>
    </div>
    <button type="submit">Submit</button>
</form>
POST /signup/email-validation returns the following
<div hx-target="this" hx-swap="outerHTML" class="error">
    <label>Email
        <input name="email"
            hx-post="/signup/email-validation"
            value="the@email.com" />
    </label>
    <div class="error-message">Email address is already taken</div>
</div>
