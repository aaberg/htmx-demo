Random number example

Now featuring a button!

Demonstrates multiple triggers and hx-target

html with htmx attributes

<div>Random number</div>
<div id="random_number"></div>
<button
    hx-get="/randomnumber"
    hx-trigger="load, click"
    hx-target="#random_number"
>New random number</button>
GET /randomnumber returns the following result

<div>${rand.next()}</div>

Random number
